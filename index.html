Como funciona

Cada fase é composta por 4 blocos progressivos: você começa pensando como usuário, depois aprende a lógica, depois entende o fluxo do sistema — e só então escreve código.

  • 👤 Fluxo de UX — o que o usuário vê e faz
  • 🧠 Lógica Básica — o que o programa precisa pensar
  • ⚙️ Fluxo de Funcionamento — o que o código faz por trás
  • 💻 Código — Python na prática
